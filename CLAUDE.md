# CLAUDE.md

## Commit Messages

Always use [Conventional Commits](https://www.conventionalcommits.org/):

```
<type>(<optional scope>): <short description>
```

Common types: `feat`, `fix`, `chore`, `refactor`, `docs`, `style`, `test`, `perf`

Examples:
- `fix(rag): replace LLMChainExtractor with EmbeddingsFilter`
- `chore: upgrade model to gemini-2.5-flash`
- `feat(api): add streaming chat endpoint`
